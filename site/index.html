About

A Wireless Local Area Network (WLAN) operating in the 6 GHz spectrum is proposed. The expected product is a radio system which operates between 5-7.125 GHz, sending packets.The access point will use Dynamic Frequency Selection (DFS) to avoid interference with pre-existing radar channels on the 6 GHz band. Included in the system package will be an antenna system.
